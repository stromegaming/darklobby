STEP 01: Build Website

Create a folder and an index.html file.

<!DOCTYPE html>
<html>
  <body>
    <h1>Site Live On GitHub</h1>
  </body>
</html>

STEP 02: Install Git

Download Git and verify with git --version.

STEP 03: Git Init

git init

STEP 04: Add Files

git add .

STEP 05: Commit

git commit -m "Initial commit"

STEP 06: Connect & Push

git remote add origin YOUR_REPO_URL
git branch -M main
git push -u origin main

STEP 07: Deployment

Go to Settings > Pages on GitHub and set branch to Main.
